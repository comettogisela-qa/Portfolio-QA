# Sobre mí

Soy QA Engineer con experiencia en análisis funcional, testing manual y automatización de pruebas para plataformas e-commerce.

Me especializo en comprender el negocio antes de comenzar las pruebas, diseñando estrategias de testing que permitan detectar riesgos tempranamente y asegurar entregas de calidad.

He trabajado en proyectos sobre SAP Commerce (Hybris), OutSystems y plataformas B2B/B2C colaborando con Product Owners, Developers y clientes.
